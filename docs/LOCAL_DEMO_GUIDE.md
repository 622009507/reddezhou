# 本地Demo使用指南

## 🎯 Demo简介

红德州本地Demo是一个基于HTML的静态展示页面，无需部署服务器，可以直接在浏览器中打开查看完整的产品展示。

## 🚀 快速开始

### 方法1：直接打开（推荐）

1. **下载项目**
   ```bash
   git clone https://github.com/622009507/reddezhou.git
   cd reddezhou
   ```

2. **打开Demo**
   - 找到项目根目录的 `demo.html` 文件
   - 双击打开
   - 在浏览器中查看

### 方法2：使用本地服务器

1. **安装Python**
   ```bash
   # Windows
   python --version
   ```

2. **启动本地服务器**
   ```bash
   # Python 3
   python -m http.server 8000

   # Python 2
   python -m SimpleHTTPServer 8000
   ```

3. **访问Demo**
   - 打开浏览器
   - 访问：http://localhost:8000/demo.html

## 🔧 自定义Demo

### 修改标题
```html
<header>
    <h1>🎴 红德州</h1>
    <p>完整的在线德州扑克游戏平台 - 微服务架构 + Netty + Unity</p>
</header>
```

### 添加截图
```html
<div class="screenshot">
    <img src="assets/screenshots/backend/your-image.png" alt="图片描述">
    <div class="screenshot-caption">图片标题</div>
</div>
```

### 修改颜色
```css
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```
